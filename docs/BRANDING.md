# OXID — Branding

- **Name:** always written **OXID** (all uppercase) in the UI wordmark and product surfaces.
  (Package/technical identifiers stay lowercase `oxid`: UCI config, service, CLI, paths.)
- **Wordmark font:** **Poppins Thin** (weight 100). Bundled locally (woff2) with
  `luci-app-oxid` — never loaded from a CDN, so the panel works fully offline on the router.
- **Usage:** the wordmark "OXID" in Poppins Thin appears in the panel header/menu brand.
  Body/UI text uses the LuCI/system font stack; Poppins Thin is reserved for the brand mark
  (and optionally large display headings).
- **Tagline:** "The best polished OpenWRT v2ray experience."
